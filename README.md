# NetSim
Projekt zaliczeniowy przedmiotu Zaawansowane Programowanie Obiektowe.

### W celu uruchomienia projektu należy wykonać następujące czynności:
1. Pobrać zawartość repozytorium na dysk lokalny.
2. Z repozytorium [googletest](https://github.com/google/googletest/releases) pobrać kod źródłowy jako archiwum .zip.
3. Rozpakować archiwum googletest do folderu nadrzędnego do głównego. Zmienić nazwę folderu googletest na **googletest-master** Struktura projektu powinna wyglądać analogicznie do:
    
- root
    - googletest-master/
    - NetSim
        - include/
        - src/
        - test/
        - README.md
        - CMake files and other files
